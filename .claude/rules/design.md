# Design Rules

- RTL layout is mandatory — verify all Tailwind directional classes
- Use text-right as default text alignment
- Color palette should be consistent — warm tones for the book theme
- Mobile responsive: test layout at 320px, 768px, 1024px breakpoints
- Images should have max-width: 100% and proper aspect ratios
- Buttons and links need hover states
- Sections should have clear visual separation
