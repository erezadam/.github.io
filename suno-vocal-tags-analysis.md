## 1. תקציר מנהלים

ברירת המחדל של Suno מייצרת זמר/ת ש"שר בעוצמה זהה" מההתחלה ועד הסוף — הפיץ' והמלודיה משתנים, אבל **המאמץ הפיזי והעוצמה של הקול נשארים שטוחים**. זה מה שגורם לרוב השירים שנוצרים ב-AI להישמע רובוטיים וחד-ממדיים.

הפתרון שהסרטון מציג: **Vocal Register Tags** (תגיות מִשלָב ווקאלי). אלו לא תגיות סגנון ולא תגיות ז'אנר — הן קטגוריה נפרדת שאומרת לזמר/ת **איך פיזית להשתמש בקול** בכל קטע של השיר. כך בונים "קשת ביצוע" (performance arc) שנעה מפגיעות עדינה ועד התפרצות רגשית.

## 2. שמונה המִשלבים ש-Suno מזהה

### ארבעת הליבה (מכסים ~90% מהצרכים)

| תגית | תיאור | מתי להשתמש |
|------|--------|------------|
| **Falsetto** | קול ראש קליל, אוורירי, "מרחף" מעל הטראק, כמעט חסר משקל | אינטרו אתרי, גשר פגיע, רגעים עדינים |
| **Chest voice** | קול חזה מלא, חם, נוכח — "קול הדיבור-שירה" | קופלאים/סיפור, רגעי בהירות וביטחון |
| **Mixed voice** | גשר בין קול חזה לרגיסטרים גבוהים; משלב חום עם טווח | בנייה והתרוממות, פְּרֵה-קוֹרוּס |
| **Belted** | דחיפת הקול לקצה הגבול תוך שמירה על שליטה — הצליל ה"מתפוצץ" | קוֹרוּסים אנתמיים, שיאים, hook ענק |

### ארבעה משלימים (סיטואציוניים)

| תגית | תיאור | הערה |
|------|--------|------|
| **Head voice** | גבוה וטהור, דומה לפלסטו אך עם יותר גוף | — |
| **Whispered** | לחישה, נשיפתי, אינטימי | הכי קשה לקבל בעקביות — להשתמש בזהירות |
| **Spoken word** | דיבור ממש — קטעי דיבור, אינטרו, אינטרלוד | — |
| **Raspy** | מחוספס, צרוד, טקסטורה גסה | גְריט רגשי, רגעים רוק-יים |

## 3. שלוש ההדגמות מהסרטון

1. **Falsetto → Belted** (Dark Pop / Emo Pop): הניגוד הדינמי המקסימלי. הקופלא מתויג `falsetto, fragile` (קל, אוורירי, כמעט נשבר), הקוֹרוּס מתויג `belted, explosive` — והמילים נכתבו ב-CAPS כדי להדגיש את המעבר הרגשי.
2. **Chest → Falsetto** (Smooth Soul R&B): הכיוון ההפוך — מתחילים מקורקעים בקול חזה (סיפור פנים-אל-פנים) ומתרוממים לקול קליל ואוורירי בקוֹרוּס (כמו זיכרון שמרחף).
3. **מגה-פרומפט: כל ארבעת הליבה** (Modern Country Ballad): קשת רגשית מלאה —
   `Falsetto (intro) → Chest (verse) → Mixed (pre-chorus) → Belted (chorus + final chorus)`.

## 4. מתי הטריק נשבר (מלכודות להימנע)

- **טמפו מהיר:** ב-BPM גבוה לזמר/ת אין זמן לעבור משלב — הם יחזרו לברירת המחדל ו"ירוצו" על המילים.
- **מילים צפופות:** אם דחסת 30 הברות בשורה קצרה, הזמר/ת עסוק/ה מדי בהגייה כדי לשנות את אופן ההפקה. תן/י מרחב: פחות הברות, תווים ארוכים יותר.
- **שימוש יתר:** החלפת משלב כל שתי שורות → קול שנשמע ערטילאי וגליצ'י. היצמד/י ל-2–3 שינויים בכל השיר; אחד לכל קטע מרכזי מספיק.
- **קונפליקט ז'אנר:** אם הסגנון הוא `soft lofi piano` אבל מתייגים קוֹרוּס כ-`belted maximum intensity` — מבקשים שני דברים מנוגדים. ה-AI יתבלבל, יתעלם מהתגית או ייתן תוצאה מבולגנת.

## 5. תובנות מעשיות — איך להשתמש בזה ב-Suno

### עקרונות מנחים
1. **שינוי בין קטעים, לא באמצע קטע.** כל קופלא = מצב ביצוע אחד; כל קוֹרוּס = מצב אחר. שינוי באמצע קטע מבלבל את המודל.
2. **בנה/י קשת ביצוע (Performance Arc).** רוב השירים הגדולים מתחילים נמוך ובונים כלפי מעלה: intro רך → verse מקורקע → pre-chorus מטפס → chorus מתפוצץ.
3. **התאם/י את המשלב לרגש של השורה:**
   - שורה פגיעה → `falsetto`
   - אמירה בטוחה → `chest voice`
   - הבנה/מתח שמטפס → `mixed voice`
   - שחרור מנצח → `belted`
4. **פשטות ואיפוק.** מעט שינויים = פגיעה חזקה כשהם קורים. ריסון הוא המפתח.

### תבנית פרקטית (להעתקה ל-Suno)

```
[Style] dark pop, emo pop, emotional, slow build, spacious

[Intro - falsetto, fragile]
(שורות עדינות, מעט הברות, תווים ארוכים)

[Verse - chest voice]
(סיפור מקורקע, ברור, שיחתי)

[Pre-Chorus - mixed voice]
(העלאת אנרגיה, טיפוס לקראת שיא)

[Chorus - belted, explosive]
(שורות ב-CAPS לשיא הרגשי — שחרור מלא)

[Final Chorus - belted, maximum intensity]
(נשארים בשיא, בלי להחזיק)
```

### צ'קליסט מהיר לפני יצירה
- [ ] בחרת את הרגע הכי פגיע ותייגת `falsetto`?
- [ ] בחרת את הקטע הכי מקורקע ותייגת `chest voice`?
- [ ] בחרת את השחרור הרגשי הגדול ותייגת `belted`?
- [ ] לא יותר מ-2–3 שינויי משלב בכל השיר?
- [ ] הטמפו והצפיפות מאפשרים מרחב למעברים?
- [ ] אין קונפליקט בין הסגנון הכללי לתגית המשלב?

## 6. נקודות חשובות
- **התגיות חייבות להיכתב באנגלית** ובתוך כותרת הקטע (למשל `[Chorus - belted, explosive]`), כפי שמודגם בסרטון.
- אפשר לחזק את האפקט עם **CAPS במילים** בקטעי השיא.
- הטכניקה רלוונטית במיוחד לבלדות וז'אנרים רגשיים (country ballad, R&B, emo pop) שמשאירים מרחב דינמי.
